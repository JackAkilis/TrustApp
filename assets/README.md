# Assets Folder

This folder contains all static assets used in the application.

## Structure

- `images/` - Image files (PNG, JPG, SVG, etc.)
- `icons/` - Icon files
- `fonts/` - Custom font files (TTF, OTF)

## Usage

To use assets in your Flutter code:

```dart
Image.asset('assets/images/example.png')
```

Make sure to add the asset path in `pubspec.yaml` under the `flutter: assets:` section.
